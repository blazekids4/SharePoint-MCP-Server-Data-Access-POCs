## ODSP Remote MCP Server

OneDrive and Sharepoint Remote MCP Server. This server connects to your tenant-specific SharePoint Online MCP server. All tools supporting OneDrive/Sharepoint files integration exposed by the ODSP MCP endpoint will be automatically discovered and made available through this server.
